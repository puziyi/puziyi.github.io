fcitx 闪退

2020-02-04 Unix/Linux/FreeBSD

今天突然用不上中文输入法，能够重启电脑或重启fcitx框架，且重启之后也能切换输入法，但是切到google pinyin开始输入立刻闪退，查看进程发现是dbus-deamon和fcitx-dbus-watcher被killed掉了，终端启动fcitx出现Warning：DBus Service Already Exists，搜索了一下很多人都是建议重装fcitx-frontend-gtk2和fcitx-frontend-gtk3，用fcitx-diagnose查看也是提示找不到gtk2。

最终解决的操作：

  • rm -rf /.config/fcitx
  • fcitx -d(这时候仍旧warning-DBus Service Already Exists)
  • sudo apt-get install --reinstall fcitx-frontend-gtk2 fcitx-frontend-gtk3(重装完重启fcitx仍旧闪退)
  • 重启，正常启动、切换输入法、输入没有出现闪退
