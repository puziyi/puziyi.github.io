[ Ecclesiastes 1：9 ] The thing that hath been, it is that which shall be; and that which is done is that which shall be done: and there is no new thing under the sun.

这粗略一看大概能知道是什么意思，但是这里为什么可以像 it is that which shall be 这样把 that which 来连用，这又并不是乱从什么地方看到的一句塑料英文，想来是我功力还不够深厚，我觉得外国人有个好的地方是，他们很会问一些稀奇古怪的问题，而且回答的人也很友善。

第二分句里面的“that”算是一个正常的代词，指代前面的名词短语，简单粗暴地看就是“the thing”，而“which”才是真的关系代词，跟第一分句里面的“that”起一样的作用，至于“it”，则是代前面整个“the thing that hath been”，第三个分句则是更为精简了这一指代的写法。这一说法也得到英语老师的支持（来自JinXiao的求证信息渠道），其指出“语法上是没有问题的，它是一个先行词 + 关系代词的结构，多见于圣经诗歌等古英文文本中”。

【 圣经 - 传道书 - 1.9 】 已有的事，后必再有；已行的事，后必再行。日光之下，并无新事。

面对近期全球的疫情蔓延和今天美联储的“0利息 + 7000亿 QE”组合拳引发的美股期指熔断事件时，很多人在说这句话，所以查了下出处。不过更有意思的是下面这句，总觉得与《道德经》中所言，遥遥呼应，言语万般，情致无异。

[ Ecclesiastes 1：7 ] All the rivers run into the sea; yet the sea is not full; unto the place from whence the rivers come, thither they return again.

【 道德经 之 四 】道冲而用之或不盈。渊兮似万物之宗。
