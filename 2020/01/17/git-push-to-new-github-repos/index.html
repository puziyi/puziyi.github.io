很多时候都是把代码clone到本地之后再做一些事情，今天突然想了想，本地库是要怎么推到github上去来着呢，origin是怎么指定的？于是建了一个demo仓库用来尝试，稍微记录一下。

基本操作顺序

一个坑点是如果新建时候点了Initialize this repository with a README的话，远程仓库不为空，README文件并不在本地仓库目录中，会报下面的错。这时或许可以重新创建一个真空库当然也可以按照hint写的这样，先把远程仓库pull下来 git pull --rebase origin master与本地合并之后再push上去。我本地也是有个README，当时没细想，觉得好像拉下来还要手动合一下，就直接点删除重新弄了个库，不过讲道理后者是更为合理普适的方法。
