In the gota-dataframe, we can easily get the numerical value in the dataframe. We can use the function (Series) Float to parse a []float64 type column of a certain dataframe, but we cannot similarly get non-number.

If the compiler runs the code like (Series) String, it returns an unindexable string like [C, C++, Golang, Python] (yeah, including the pair of square brackets), according to the definition of it.

So, how can I get the Golang string only, without square brackets or other elements?

Since the function (DataFrame) Col returns value in series.Series type, elements can be returned by function (Series) Elem, which would return the element of a series for the given index(and be careful with the panic of “out of bounds”).
