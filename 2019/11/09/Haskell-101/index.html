周五被科普安利了一波函数式编程和Haskell语言，它是一个purely functional programming language，不同于之前经常接触到的imperative programming languages (Golang, C/C++, Python …)，它并不是写出一系列的操作和命令交给计算机去完成，而是去“定义”。简单写两句代码对比作为例子，比如“阶乘”这个概念，在IPL里面是描述历遍小于n的值，并将它们逐个乘起来这样一个操作，而在PFL里面是定义说，“阶乘”是1到n的乘积。
