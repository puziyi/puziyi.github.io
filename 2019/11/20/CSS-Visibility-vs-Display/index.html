CSS Visibility vs Display

记岔了CSS中display和visibility属性，写了一个display: hidden;的样式没起作用，特意查了一下，且简记之。

display: none;是隐藏元素，表现为页面上这个元素不可见，且该元素不会占据页面中的空间，是真的隐身了；

visibility: hidden;是设置元素不可见（但元素没有被隐藏），仅仅表现为元素在页面上不可见，是“捂住眼睛看不到我”式的躲藏。
