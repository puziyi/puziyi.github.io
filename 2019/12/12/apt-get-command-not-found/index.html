问题描述： 换了台云机器，安装时候运行apt install xxx和apt-get install xxx时候都返回错误提示“apt-get: command not found”

问题原因：查了一下说是有可能这台机器不是Debian-based distro，而是CentOS-based/Fedora-based distro，用命令cat /etc/*-release看了下，的确是CentOS Linux

解决方法：use yum instead of apt/apt-get

Ref at stackoverflow

One more： apt 是在 Ubuntu 14 / Debian 8 才开始使用，之前的版本中为apt-get
