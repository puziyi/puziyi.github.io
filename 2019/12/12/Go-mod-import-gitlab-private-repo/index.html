问题描述： go run main.go之后，一直让输入Gitlab账户密码，没有进入到主程当中。

问题诱因：

  • Golang版本为1.13（go1.13 special）
  • 使用 go mod import gitlab private repo
  • go mod 中没有正确 replace 路径
  • go get 不支持 gitlab 的 subgroup

解决方案：

把私有库clone到本地$GOPATH/src目录下，在项目go.mod中添加replace gitlab.com/username/reponame => /src/reponame的引用说明。
