Git not found in VS Code

VS Code Warning Pop-up says: Git not found. Install it or configure it using ‘git.path’ setting. And the built-in Git extension fails to work.

Solution:

  • First, find the path of git using whereis command
  • Then open File-Preferences-Setting in VS Code, search ‘git.path’, change it from “null” to the valid path we have found in the above step. The relative item in settings.json might like this: {"git.path": "/usr/local/git/bin/git"}
  • Reload editor window and the extension should work well
This article was last updated on days ago, and the information described in the article may have changed.
