常用任务切换控制命令

  • command & 命令后台执行
  • Ctrl + d 二进制值EOF，结束终端输入
  • Ctrl + z Sent SIGTSTP signal，to terminal stop；挂起任务，可通过bg或fg继续执行

  • Ctrl + c Sent SIGINT signal，to interrupt the process；终端强制结束任务执行

  • jobs 查看当前任务

  • bg 将任务切换到后台运行，但不接受终端输入；%i 指定对应任务
  • fg 将后台任务切到前台；%i 指定对应任务
  • kill 结束任务；%i 指定对应任务
